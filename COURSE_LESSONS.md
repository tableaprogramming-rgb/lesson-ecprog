# PROG2: Computer Programming 2 - Course Lessons

## Course Overview
- **Course Number:** PROG2
- **Course Title:** Computer Programming 2
- **Course Units:** 3.0
- **Pre-requisite:** Computer Programming 1 (PROG 1)
- **Term Offered:** First Year - Second Semester
- **Institution:** University of San Jose-Recoletos

## Course Description
This is an advanced course in C programming for students with a solid understanding of C programming basics. The course prepares students to master the complexity of writing their own functions, covering multi-dimensional arrays, pointers, strings, structs, and abstraction. Students will learn to optimize code for performance and apply principles of good software design.

## Course Learning Outcomes (CILO)
1. Simulate the behavior and detect the output of program codes involving fundamental programming constructs
2. Choose appropriate conditional and iteration constructs for a given programming task
3. Apply techniques of structured (functional) decomposition to break a program into smaller pieces
4. Describe the mechanics of parameter passing and issues associated with scoping
5. Design, implement, test, and debug a program using fundamental programming constructs (computation, I/O, conditionals, iterations, functions)
