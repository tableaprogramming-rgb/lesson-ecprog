Welcome!

Enrichment Course for Computer Programming is an advanced summer course in C programming for students with a solid understanding of C programming basics. This course prepares you to master the complexity of writing your own functions, covering multi-dimensional arrays, pointers, strings, structs, and abstraction.

You will learn to optimize code for performance and apply principles of good software design. By the end of this course, you'll be able to develop robust C programs that use sophisticated data structures and programming techniques.

Course Learning Outcomes

By the end of this course, you will be able to:

  • ✓ Simulate program codes and detect output of fundamental programming constructs
  • ✓ Choose appropriate conditional and iteration constructs for programming tasks
  • ✓ Apply structured (functional) decomposition to break programs into smaller pieces
  • ✓ Describe mechanics of parameter passing and scoping issues
  • ✓ Design, implement, test, and debug programs using fundamental constructs

All Lessons

1

Programming Control Structures

Topics: Selection, Repetition

Develop programs using selection and repetition statements. Code simulation and evaluation.

2

Functions

Topics: Void Functions, Value-Returning Functions, Parameter Passing, Recursion

Differentiate parameter passing methods. Implement modularization for code maintainability.

3

Single-Dimensional Arrays

Topics: Array Manipulation

Declare arrays appropriately. Evaluate code and develop array manipulation programs.

4

Multi-Dimensional Arrays

Topics: Array Processing, Arrays as Function Parameters

Develop programs using 2D arrays as parameters. Array processing and manipulation.

5

Pointers

Topics: Pointer Variables, Expressions with Pointers

Determine output of pointer expressions. Design and develop programs using pointers.

6

Strings

Topics: Pre-defined String Functions, String Manipulation

Evaluate code with strings. Develop programs that manipulate strings as parameters.

7

Structs

Topics: Struct Definition, Array of Structs

Declare struct elements appropriately. Develop programs with struct arrays and data manipulation.

8

Abstraction

Topics: Procedural Abstraction, Data Abstraction

Create function libraries. Implement data encapsulation and abstraction concepts.
