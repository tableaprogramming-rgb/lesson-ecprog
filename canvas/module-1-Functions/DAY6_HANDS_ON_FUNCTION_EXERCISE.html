Proof of Concept: Grade Calculator System

POC

Accurate Grade Calculator for Teachers

👩‍🏫 Meet Elena Rodriguez, 28 — High School Computer Science Teacher

"I'm grading 120 students across three classes. Every semester, I manually calculate quiz and exam averages, then compute final grades using a weighted formula. I've caught several calculation errors after reporting grades, which creates confusion with students and parents. I need a quick, reliable tool to calculate accurate grades without mistakes."

🎯 Success Metrics:

  • ✓ Calculate 100% accurate weighted grades (40% quizzes + 60% exams)
  • ✓ Generate correct letter grades (A/B/C/D/F) every time
  • ✓ Process one student's grades in under 30 seconds
  • ✓ Eliminate manual calculation errors

📱 Typical Interaction:

  1. Teacher runs the program and enters a quiz score (85)
  2. Program asks for an exam score (92)
  3. Program calculates and displays:
    • Quiz Score: 85.00
    • Exam Score: 92.00
    • Final Grade: 89.20
    • Letter Grade: B
  4. Teacher can confidently report accurate grade to student

⚙️ What You'll Implement:

Write 5 functions to solve Elena's problem:

  1. 1. A function to input a single score
    Takes: a label describing what score is being entered (e.g., "Quiz" or "Exam")
    Does: Asks the user to enter a score and waits for their response
    Returns: The score the user entered as a number
  2. 2. A function to calculate a weighted final grade
    Takes: a quiz score and an exam score
    Does: Multiplies quiz by 0.4 (40%) and exam by 0.6 (60%), then adds them together
    Returns: The final grade as a decimal number
  3. 3. A function to determine a letter grade
    Takes: the final numeric grade
    Does: Checks which range the grade falls into (90+, 80+, 70+, 60+, below 60)
    Returns: The corresponding letter (A, B, C, D, or F)
  4. 4. A function to display the grade report
    Takes: quiz score, exam score, final grade, and letter grade
    Does: Prints a formatted report showing all the information nicely
    Returns: Nothing (just prints)

Note: That's 4 functions, not 5! This is a simpler version focused on learning functions without arrays.

Function Requirements

1️⃣ Input Score Function

What it does: Asks the user to enter one score

Input parameter needed:

  • A label describing the score (e.g., "Quiz" or "Exam")

Returns: The score value that the user entered

Example behavior:

Enter Quiz Score (0-100): 85

2️⃣ Calculate Final Grade Function

What it does: Combines a quiz score and an exam score using weighted percentages

Input parameters needed:

  • The quiz score
  • The exam score

Returns: The final grade as a decimal number

Calculation: (quiz score × 0.4) + (exam score × 0.6)

Quiz counts for 40%, Exam counts for 60%

3️⃣ Get Letter Grade Function

What it does: Converts a numeric grade to a letter grade

Input parameter needed:

  • The final numeric grade (from function 2)

Returns: The letter grade (A, B, C, D, or F)

Grading ranges:

Score Range Letter Grade
90 to 100 A
80 to 89 B
70 to 79 C
60 to 69 D
Below 60 F

4️⃣ Display Report Function

What it does: Prints out a formatted grade report showing all values

Input parameters needed:

  • The quiz score
  • The exam score
  • The final numeric grade
  • The final letter grade

Returns: Nothing (just prints to screen)

Example output should look like:

========== GRADE REPORT ==========
Quiz Score:       85.00
Exam Score:       92.00
Final Grade:      89.20
Letter Grade:     B
==================================

Program Flow (How main() should work)

Your main function should follow this sequence:

  1. Call the input function with "Quiz" to get the quiz score from the user and store it
  2. Call the input function with "Exam" to get the exam score from the user and store it
  3. Call the final grade function with both scores to calculate the weighted final grade
  4. Call the letter grade function with the final grade to determine the letter (A/B/C/D/F)
  5. Call the display function with all four values to show Elena's complete grade report

Remember: Your main function simply calls the functions in order. Each function does its specific job, and you use the results from one function as input to the next.
