Key Insight

Loop Accumulation:

Use a variable (sum) to collect results from each iteration.
sum = sum + value adds to the total

In this exercise:
We accumulated: 1 + 2 + 3 + 4 = 10

✓ Track TWO variables in loops: the control variable (i) and the accumulator (sum)
