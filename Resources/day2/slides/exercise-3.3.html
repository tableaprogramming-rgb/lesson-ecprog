        x = x - 2;

}

Key Insight

Decrement Loops:

Loops can go backwards: x = x - 2 instead of x = x + 1

In this exercise:
Started at 5, went down by 2: 5 → 3 → 1 → -1 (stop)

✓ The condition check (x > 0) determines when the loop stops
