Key Takeaway

While Loop Mechanics:

  1. Check condition BEFORE each iteration
  2. If TRUE: execute loop body
  3. Go back to step 1
  4. If FALSE: exit loop

Remember: The condition is checked BEFORE each iteration, not after!
