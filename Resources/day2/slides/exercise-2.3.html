Exercise 2.3: Complete Code

#include <stdio.h>

int main() {
    int day = 6;  // 1=Mon...7=Sun

    if (day == 6 || day == 7) {
        printf("It's the weekend!\n");
    } else {
        printf("It's a weekday\n");
    }

    return 0;
}

Key Insight

OR Operator (||):

For OR to be TRUE, AT LEAST ONE part must be TRUE.

In this exercise:
TRUE || ? = TRUE (we don't even check the second part)

✓ Compare with AND: AND needs both true; OR only needs one
