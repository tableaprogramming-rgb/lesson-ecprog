Exercise 4.1: Complete Code

#include <stdio.h>

int main() {
    int i = 1;

    while (i <= 4) {
        if (i % 2 == 0) {
            printf("%d is even\n", i);
        } else {
            printf("%d is odd\n", i);
        }
        i = i + 1;
    }

    return 0;
}

Key Insight

Mixed Logic (Loop + Condition):

The if-else INSIDE the loop runs with EVERY iteration.
Different iterations can take different branches!

In this exercise:
Even numbers go to if; odd numbers go to else

✓ Always trace BOTH the loop condition AND the if-else condition
