Exercise 4.2: Complete Code

#include <stdio.h>

int main() {
    int count = 0;
    int i = 1;

    while (i <= 5) {
        if (i > 2 && i < 5) {
            count = count + 1;
        }
        i = i + 1;
    }

    printf("Count: %d\n", count);

    return 0;
}

Step 2-6: Loop Iterations

Check i > 2 && i < 5 each iteration

Only i=3 and i=4 satisfy BOTH conditions

// Iteration 1: i=1
1 > 2? NO → FALSE
→ skip count

// Iteration 2: i=2
2 > 2? NO → FALSE
→ skip count

// Iteration 3: i=3
3 > 2? YES, 3 < 5? YES
→ TRUE, count = 1

// Iteration 4: i=4
4 > 2? YES, 4 < 5? YES
→ TRUE, count = 2

// Iteration 5: i=5
5 > 2? YES, 5 < 5? NO
→ FALSE, skip count

Key Insight

AND in Loops with Accumulators:

Use AND to select which iterations increment an accumulator.
Only iterations where BOTH conditions are TRUE increment count.

In this exercise:
Counted values in range (2,5): i=3 and i=4 (count=2)

✓ This is a common pattern: count items matching a range/condition
