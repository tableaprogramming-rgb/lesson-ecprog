Exercise 2.1

Simple AND Condition

Learning: AND operator basics

Both parts must be TRUE for AND to be true

The Code

#include <stdio.h>

int main() {
    int age = 20;
    int hasLicense = 1;

    if (age >= 18 && hasLicense == 1) {
        printf("You can drive\n");
    } else {
        printf("You cannot drive\n");
    }

    return 0;
}

What will this program output?

Key Takeaway

AND Operator (&&):

For AND to be TRUE, BOTH parts must be TRUE.

age >= 18 && hasLicense == 1
TRUE && TRUE = TRUE

✓ If either part was FALSE, the whole condition would be FALSE
