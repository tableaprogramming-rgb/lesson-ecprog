Nested Loops (Challenge Level)

Exercise 1.5: Complete Code

#include <stdio.h>

int main() {
    int total = 0;

    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 2; j++) {
            total = total + 1;
        }
    }

    printf("Total: %d\n", total);

    return 0;
}

Understanding Nested Loops

This code has:

  • Outer Loop: for (int i = 1; i <= 3; i++) runs 3 times
  • Inner Loop: for (int j = 1; j <= 2; j++) runs 2 times per outer iteration
  • Total Executions: 3 × 2 = 6 times
  • Order: Inner loop completes before outer loop advances

KEY: Each time outer loop runs once, inner loop runs fully!

Key Concepts to Remember

  • Nested Loops: Loop inside another loop
  • Execution Order: Inner loop runs completely before outer loop advances
  • Total Iterations: Outer count × Inner count = Total
  • Variable Scope: Inner loop variables reset each iteration
  • Complex Tracing: Track both i AND j values at each step
