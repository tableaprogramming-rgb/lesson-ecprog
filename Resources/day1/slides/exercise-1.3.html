Key Concepts to Remember

  • For Loop Structure: for (init; condition; increment) { body }
  • Loop Execution: Runs while condition is TRUE, stops when FALSE
  • Loop Counter: Updated with i++ after each iteration
  • Accumulation: sum = sum + i adds current i value to previous sum
  • Trace Every Iteration: Don't skip steps—update variables each time
