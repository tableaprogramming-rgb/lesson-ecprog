Key Concepts to Remember

  • If-Else-If Chain: Tests conditions in order from top to bottom
  • First TRUE Wins: Executes the first TRUE condition's block
  • No Cascading: Once a condition is TRUE, remaining conditions are skipped
  • Multiple Conditions: Use >= (greater than or equal) carefully with ranges
  • Trace Every Step: Check each condition even if not executed
