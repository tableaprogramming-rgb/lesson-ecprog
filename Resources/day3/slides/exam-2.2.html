The Context

Goal:

Sum all EVEN numbers from 10 to 15

Calculation:

Even numbers: 10, 12, 14
Sum = 10 + 12 + 14 = 36

The Buggy Code

This code won't even compile!

#include <stdio.h>

int main() {
    int sum = 0;

    for (int i = 10; i <= 15; i++) {
        if (i % 2 = 0) {
            sum = sum + i;
        }
    }

    printf("Sum: %d\n", sum);

    return 0;
}
