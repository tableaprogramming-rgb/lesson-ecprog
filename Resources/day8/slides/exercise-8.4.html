Exercise 8.4: Functions with 2D Arrays

Passing 2D arrays as parameters

#include <stdio.h>

int sumMatrix(int matrix[3][3]) {
    int sum = 0;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            sum += matrix[i][j];
        }
    }
    return sum;
}

int main() {
    int arr[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    int total = sumMatrix(arr);
    printf("Total: %d\n", total);

    return 0;
}

Step 1: Function Parameter

Must specify both dimensions for 2D array

int sumMatrix(int matrix[3][3]) {
    int sum = 0;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            sum += matrix[i][j];
        }
    }
    return sum;
}

int main() {
    int arr[3][3] = {...};
    int total = sumMatrix(arr);
    printf("Total: %d\n", total);
    return 0;
}

Parameter Syntax

Correct:

int sumMatrix(int matrix[3][3])

Also Valid:

int sumMatrix(int matrix[][3])
(row count optional)

Why [3][3]?

Compiler needs column count to calculate memory offsets

Key Points

• Array passed by reference (no copy)
• Function modifies sum locally
• Returns single value to main
• Result printed correctly

Concept Summary

Functions with 2D arrays

1. Parameter Must Include Column Count

Row count optional, but column count REQUIRED

2. Array Passed by Reference

Function receives address, not copy

3. Function Accesses All Elements

Use nested loops like in main()

4. Return Single Value

Function returns result for use in main()
