Exercise 8.7: Practical Application

Grade Management System - Real-World 2D Arrays

#include <stdio.h>

int main() {
    // 4 students, 3 subjects (Math, English, Science)
    int grades[4][3] = {
        {85, 90, 78},
        {92, 88, 95},
        {78, 85, 82},
        {88, 92, 89}
    };

    // Calculate student averages
    printf("STUDENT AVERAGES:\n");
    for (int i = 0; i < 4; i++) {
        int sum = 0;
        for (int j = 0; j < 3; j++) {
            sum += grades[i][j];
        }
        double avg = sum / 3.0;
        printf("Student %d: %.2f\n", i, avg);
    }

    // Calculate subject averages
    printf("\nSUBJECT AVERAGES:\n");
    for (int j = 0; j < 3; j++) {
        int sum = 0;
        for (int i = 0; i < 4; i++) {
            sum += grades[i][j];
        }
        double avg = sum / 4.0;
        printf("Subject %d: %.2f\n", j, avg);
    }

    return 0;
}

Data Structure Overview

Understanding the grade matrix

Grade Matrix:
4 students × 3 subjects

grades[4][3] = {
    {85, 90, 78},
    {92, 88, 95},
    {78, 85, 82},
    {88, 92, 89}
};

Index meanings:
grades[student][subject]

Students: 0, 1, 2, 3
Subjects: 0(Math), 1(Eng), 2(Sci)

Example:
grades[0][0] = 85 (Student 0, Math)
grades[1][2] = 95 (Student 1, Science)
grades[3][1] = 92 (Student 3, English)

Student Averages Calculation

Sum each row, divide by 3 subjects

// For each student
for (int i = 0; i < 4; i++) {
    int sum = 0;
    // Sum all subjects for this student
    for (int j = 0; j < 3; j++) {
        sum += grades[i][j];
    }
    // Calculate average
    double avg = sum / 3.0;
    printf("Student %d: %.2f\n",
           i, avg);
}

Pattern: Fix row (i), loop columns (j)

Student Averages

(78+85+82)/3 = 81.67

Subject Averages Calculation

Sum each column, divide by 4 students

// For each subject
for (int j = 0; j < 3; j++) {
    int sum = 0;
    // Sum all students for this subject
    for (int i = 0; i < 4; i++) {
        sum += grades[i][j];
    }
    // Calculate average
    double avg = sum / 4.0;
    printf("Subject %d: %.2f\n",
           j, avg);
}

Pattern: Fix column (j), loop rows (i)
