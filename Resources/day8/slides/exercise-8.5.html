Exercise 8.5: Searching in 2D Arrays

Finding elements and their positions

#include <stdio.h>

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    int target = 5;
    int found = 0;
    int row = -1, col = -1;

    for (int i = 0; i < 3 && !found; i++) {
        for (int j = 0; j < 3 && !found; j++) {
            if (matrix[i][j] == target) {
                found = 1;
                row = i;
                col = j;
            }
        }
    }

    if (found) {
        printf("Found %d at [%d][%d]\n", target, row, col);
    } else {
        printf("Not found\n");
    }

    return 0;
}

Key Points

Concept Summary

Search algorithms in 2D arrays

1. Use Flag for Early Exit

&&!found in loop stops search

2. Store Position When Found

Save row and col indices for later use

3. Check Before Using Result

Verify found is true before accessing row/col

4. Linear Search Pattern

Check each element sequentially until found
