Exercise 8.2: Nested Loops & Traversal

Using nested loops to iterate through a 2D array

#include <stdio.h>

int main() {
    int arr[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    int sum = 0;
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            sum += arr[i][j];
        }
    }

    printf("Sum: %d\n", sum);
    return 0;
}

Key Concepts from Exercise 8.2

Understanding nested loop execution

1. Nested Loop Pattern

Outer loop (i) = rows, Inner loop (j) = columns
Inner loop completes ALL iterations before outer loop increments

2. Total Iterations

For 2×3 array: 2 outer × 3 inner = 6 total iterations

3. Accumulation Pattern

sum += arr[i][j] adds each element to sum as loop iterates

4. Memory Access Order

Accesses [0][0], [0][1], [0][2], [1][0], [1][1], [1][2]
Follows row-major order (matches memory layout)
