Exercise 8.3: Row and Column Operations

Calculating sums of specific rows and columns

#include <stdio.h>

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    // Sum of row 1
    int rowSum = 0;
    for (int j = 0; j < 3; j++) {
        rowSum += matrix[1][j];
    }
    printf("Row 1 sum: %d\n", rowSum);

    // Sum of column 1
    int colSum = 0;
    for (int i = 0; i < 3; i++) {
        colSum += matrix[i][1];
    }
    printf("Column 1 sum: %d\n", colSum);

    return 0;
}

Key Differences: Row vs Column

Understanding selective traversal

Row Sum (Fixed Row)

for (int j = 0; j < 3; j++)
Access: matrix[1][0], matrix[1][1], matrix[1][2]
Pattern: i is constant, j varies

Column Sum (Fixed Column)

for (int i = 0; i < 3; i++)
Access: matrix[0][1], matrix[1][1], matrix[2][1]
Pattern: j is constant, i varies

Key Insight

Fix the index you don't want to change,
Loop through the index you want to vary

Concept Summary

Row vs Column operations

1. Row Operations Use Single Loop

Only loop columns; row index is fixed

2. Column Operations Use Single Loop

Only loop rows; column index is fixed

3. More Efficient Than Nested Loops

For single row/column: 3 iterations instead of 9

4. Same Accumulation Pattern

Initialize variable, add elements in loop, use result
