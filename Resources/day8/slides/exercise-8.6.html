Exercise 8.6: Matrix Transpose

Swapping rows and columns

#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    int transpose[3][2];

    // Transpose: swap indices
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            transpose[j][i] = matrix[i][j];
        }
    }

    // Print results...
    return 0;
}

Transpose Operation

Original 2×3 becomes 3×2

Original:  2 rows × 3 cols
    matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

Transpose: 3 rows × 2 cols
    transpose[3][2]

Key Operation:
transpose[j][i] = matrix[i][j];

Index Swap:
matrix[0][0] → transpose[0][0]
matrix[0][1] → transpose[1][0]
matrix[0][2] → transpose[2][0]
matrix[1][0] → transpose[0][1]
matrix[1][1] → transpose[1][1]
matrix[1][2] → transpose[2][1]

Visual Transformation

Transpose (3×2):

Rows become Columns!

Key Concept: Index Swapping

The secret to transpose

Original Access

matrix[i][j]
i = row index, j = column index

Transposed Access

transpose[j][i]
Indices swapped! Rows ↔ Columns

Dimension Change

Original: [rows][cols]
Transpose: [cols][rows]
2×3 → 3×2, 3×4 → 4×3, etc.

Concept Summary

Matrix transpose operation

1. Swap Row and Column Indices

Use transpose[j][i] = matrix[i][j]

2. Create New Array with Swapped Dimensions

If original is [m][n], transpose is [n][m]

3. Loop Bounds Must Match Original

Loop through original's rows and columns

4. Real-World Application

Linear algebra, matrix math, rotations
