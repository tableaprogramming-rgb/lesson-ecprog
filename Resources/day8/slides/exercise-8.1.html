Exercise 8.1: 2D Array Declaration & Initialization

Understanding array structure and accessing elements

#include <stdio.h>

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    printf("matrix[1][2] = %d\n", matrix[1][2]);
    printf("matrix[2][1] = %d\n", matrix[2][1]);

    return 0;
}

Key Concepts from Exercise 8.1

What you should understand

1. Array Declaration

int matrix[3][3] = 3 rows, 3 columns = 9 total elements

2. Initialization

Nested braces organize data by row (outer brace) and columns (inner braces)

3. Indexing

matrix[row][col] - Both indices required, 0-indexed (0 to 2 for a 3×3)

4. Memory Layout

Row-major order: [0][0], [0][1], [0][2], [1][0], [1][1], [1][2], [2][0], [2][1], [2][2]
