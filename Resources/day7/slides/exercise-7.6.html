Step 7: Final Iteration & Return

Last iteration (i=3), then exit loop and return

temps[3] = 70

Condition 4 < 4 is FALSE

Loop exits when condition becomes false; program returns 0 (success)

Summary: Array Traversal with Loop

What We Learned:
  • for(int i = 0; i < 4; i++) iterates 4 times (i = 0,1,2,3)
  • Loop counter i is used as array index: temps[i]
  • Each iteration accesses the next element in the array
  • Loop automatically increments i with i++
  • Loop stops when condition i < 4 becomes false
  • Traversal avoids writing same code 4 times!
