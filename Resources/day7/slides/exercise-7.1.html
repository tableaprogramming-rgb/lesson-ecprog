Summary: Array Declaration & Initialization

What We Learned:
  • int numbers[5] creates space for 5 integers
  • = {10, 20, 30, 40, 50} initializes with values
  • Arrays use 0-based indexing (0, 1, 2, 3, 4)
  • Index = position - 1
  • Last valid index = size - 1

Last: 50
