Summary: User Input & Output with Loops

What We Learned:
  • int numbers[3] declares empty array of 3 integers
  • First loop: scanf("%d", &numbers[i]) reads values into array
  • & (address-of) operator gives memory address of array element
  • Loop counter i indexes into the array for both input and output
  • Array fills dynamically as user provides input
  • Second loop: Use printf with loop to display all values
  • Loops make code cleaner and more scalable than hardcoding indices
